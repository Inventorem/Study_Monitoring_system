# Study monitoring system


Как развернуть проект
```
git clone в нужную директорию
```
```
shell1 :
npx hardhat clean
npx hardhat node
```
```
shell2 :
npx hardhat run scripts \front_deploy.js --network localhost
```

Прогнать тесты
```
git clone в нужную директорию
```
```shell
npx hardhat test
```
